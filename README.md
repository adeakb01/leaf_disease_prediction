# Plant Disease Detection System for Sustainable Agriculture
An interactive web application built with Streamlit and TensorFlow that uses deep learning to detect plant diseases from leaf images. The system can identify three tomato leaf conditions: Early Blight, Late Blight, and Healthy leaves.

## Features
-**Disease Recognition**: Upload an image of a tomato leaf to get instant disease classification

-**High-Accuracy Model**: Custom-built CNN model achieving ~99% training accuracy and ~94% validation accuracy

-**User-Friendly Interface**: Clean Streamlit interface with sidebar navigation

-**Cloud-Ready**: Model hosted on Google Drive for easy deployment

-**Real-Time Predictions**: Get immediate results with visual feedback
